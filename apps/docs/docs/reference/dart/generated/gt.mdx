gt()用于查找所有在所述`列（column）`上的值大于指定`值（value）`的记录。

## 案例教程
### 案例1 (使用select)

<Tabs
  scrollable
  size="small"
  type="underlined"
  defaultActiveId="demo2"
>

<TabPanel id="demo2" label="使用方法">

```dart
                                                                              
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .gt('country_id', 250);
```
</TabPanel>
</Tabs>
