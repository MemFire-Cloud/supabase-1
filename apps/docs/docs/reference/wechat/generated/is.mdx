仅匹配`列`值与`指定值`相等的行。



对于非boolean型列，这只与检查`column`的值是否为NULL有关。
`column`的值是NULL，通过设置`value`为`null`。

对于boolean型列，你也可以将`value`设置为`true`或`false`，它的行为与
它的行为与`.eq()`相同。


## 案例教程

### 案例1  (检查是否为 null)

<Tabs scrollable size="small" type="underlined" defaultActiveId="demo1">
<TabPanel id="demo1" label="建表">

  ```sql
                                                                              
create table
  countries (id int8 primary key, name text);

insert into
  countries (id, name)
values
  (1, 'null'),
  (2, null);
  ```

</TabPanel>
<TabPanel id="demo2" label="使用方法">

  ```ts
const { data, error } = await supabase
  .from('countries')
  .select()
  .is('name', null)
  ```

</TabPanel>

<TabPanel id="demo3" label="返回结果">

  ```ts
{
  "data": [
    {
      "id": 1,
      "name": "null"
    }
  ],
  "status": 200,
  "statusText": "OK"
}

  ```

</TabPanel>


<TabPanel id="demo4" label="注意事项">

使用`eq()`筛选器在过滤`null`时不起作用。相反，您需要使用`is()`。

</TabPanel>

</Tabs>
