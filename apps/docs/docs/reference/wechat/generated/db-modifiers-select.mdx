对查询结果执行SELECT。

默认情况下，`.insert()`, `.update()`, `.upsert()`, 和 `.delete()`不会返回修改过的记录。通过调用这个方法，修改过的行会返回到`data`。

```ts
const { data, error } = await supabase
  .from('countries')
  .upsert({ id: 1, name: 'Algeria' })
  .select()
```
