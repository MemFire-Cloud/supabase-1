gt()用于查找所有在所述`列（column）`上的值大于指定`值（value）`的记录。

## 案例教程

### 案例1 （使用select）

<Tabs scrollable size="small" type="underlined" defaultActiveId="demo1">
<TabPanel id="demo1" label="建表">

  ```sql
                                                                                
create table
  countries (id int8 primary key, name text);

insert into
  countries (id, name)
values
  (1, 'Afghanistan'),
  (2, 'Albania'),
  (3, 'Algeria');
  ```

</TabPanel>
<TabPanel id="demo2" label="使用方法">

  ```ts
                                                                                
const { data, error } = await supabase
  .from('countries')
  .select()
  .gt('id', 2)
  ```

</TabPanel>
<TabPanel id="demo3" label="返回结果">

  ```json
                                                                                
{
  "data": [
    {
      "id": 3,
      "name": "Algeria"
    }
  ],
  "status": 200,
  "statusText": "OK"
}
  ```

</TabPanel>

<TabPanel id="demo4" label="注意事项">

当使用[保留字](https://www.postgresql.org/docs/current/sql-keywords-appendix.html)作为列名时，需要添加双引号，例如：`.gt('"order"', 2)`

</TabPanel>


</Tabs>
