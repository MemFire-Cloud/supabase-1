from.remove()用于删除存储桶中的文件

需要RLS策略权限:
  - `buckets`表的权限: 无
  - `objects`表的权限：`delete`和`select`权限

请参考[存储指南](/docs/guides/storage#access-control)中关于访问控制的工作方式。

## 案例教程
