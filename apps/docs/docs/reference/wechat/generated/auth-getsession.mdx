getSession()用于检索会话信息，返回会话，并在必要时进行刷新。这意味着如果需要，会话会被刷新后返回。
如果未检测到会话，返回的会话可能为 `空（null）`，这可能发生在用户未登录或已退出登陆的情况下。

* 这个方法用于检索当前的本地会话（即本地存储）。
* 当会话的访问令牌过期时，该方法会使用刷新令牌来获取一个新的会话。
