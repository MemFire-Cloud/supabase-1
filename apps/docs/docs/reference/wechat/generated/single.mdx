将`数据(data)`作为单个对象返回，而不是返回一个对象数组。


## 案例教程
### 案例1 

<Tabs scrollable size="small" type="underlined" defaultActiveId="demo1">

<TabPanel id="demo1" label="建表">

  ```sql
create table
  countries (id int8 primary key, name text);

insert into
  countries (id, name)
values
  (1, 'Afghanistan'),
  (2, 'Albania'),
  (3, 'Algeria');

  ```

</TabPanel>

<TabPanel id="demo2" label="使用方法">

  ```ts
const { data, error } = await supabase
  .from('countries')
  .select('name')
  .limit(1)
  .single()
  ```

</TabPanel>


<TabPanel id="demo3" label="返回结果">

  ```json
{
  "data": {
    "name": "Afghanistan"
  },
  "status": 200,
  "statusText": "OK"
}
  ```

</TabPanel>
</Tabs>
