refreshSession()用于获取新的会话，无论会话是否已过期，都将返回一个新的会话。该方法接受一个可选的当前会话作为参数。如果未传入当前会话，那么 refreshSession() 方法将尝试从 getSession() 方法中获取当前会话。
如果当前会话的刷新令牌无效，将抛出一个错误。

* 该方法将刷新并返回一个新的会话，无论当前会话是否已过期。
