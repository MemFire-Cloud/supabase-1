listUsers()用于获取用户列表，默认每页返回50个用户。


## 案例教程

### 案例1 （获取用户的一页数据）

<Tabs scrollable size="small" type="underlined" defaultActiveId="demo2">

<TabPanel id="demo2" label="使用方法">

  ```ts
                                                                                   
const { data: { users }, error } = await supabase.auth.admin.listUsers()
  ```

</TabPanel>
</Tabs>

### 案例2 （用户的分页列表）

<Tabs scrollable size="small" type="underlined" defaultActiveId="demo2">

<TabPanel id="demo2" label="使用方法">

  ```ts
                                                                                   
const { data: { users }, error } = await supabase.auth.admin.listUsers({
  page: 1,
  perPage: 1000
})
  ```

</TabPanel>
</Tabs>
