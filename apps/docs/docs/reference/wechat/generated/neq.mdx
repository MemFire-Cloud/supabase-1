neq()用于匹配`列`值不等于`指定值`的行。

## 案例教程

### 案例1

<Tabs scrollable size="small" type="underlined" defaultActiveId="demo1">
<TabPanel id="demo1" label="建表">

  ```sql
                                                                                
create table
  countries (id int8 primary key, name text);

insert into
  countries (id, name)
values
  (1, 'Afghanistan'),
  (2, 'Albania'),
  (3, 'Algeria');

  ```

</TabPanel>
<TabPanel id="demo2" label="使用方法">

  ```ts
                                                                                
const { data, error } = await supabase
  .from('countries')
  .select()
  .neq('name', 'Albania')
  ```

</TabPanel>
<TabPanel id="demo3" label="返回结果">

  ```json
                                                                                
{
  "data": [
    {
      "id": 1,
      "name": "Afghanistan"
    },
    {
      "id": 3,
      "name": "Algeria"
    }
  ],
  "status": 200,
  "statusText": "OK"
}
  ```

</TabPanel>

</Tabs>
