ilike()用于查找所有在所述`列（column）`上的值与提供的 `模板（pattern）`相符的记录（不区分大小写）。

## 案例教程
### 案例1 (使用select)

<Tabs scrollable size="small" type="underlined" defaultActiveId="demo1">
<TabPanel id="demo1" label="建表">

  ```sql
create table
  countries (id int8 primary key, name text);
                                                                              
insert into
  countries (id, name)
values
  (1, 'Afghanistan'),
  (2, 'Albania'),
  (3, 'Algeria');

  ```

</TabPanel>
<TabPanel id="demo2" label="使用方法">

  ```ts
const { data, error } = await supabase
                                                                              
  .from('countries')
  .select()
  .ilike('name', '%Alba%')
  ```

</TabPanel>
</Tabs>
